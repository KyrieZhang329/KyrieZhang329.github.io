## 📖 什么是这个文件

这是一个个人主页的网页文件，就像是你的网上名片。它包含：
- 你的照片和基本信息（左侧）
- About Me 自我介绍
- 你的工作/学习经历
- 你的项目展示
- 你的发表论文
- 你的联系方式

## 🛠️ 需要准备什么

### 必需品：
1. **一个文本编辑器**（选择其中一个即可）：
   - 记事本（Windows自带，但不推荐）
   - **Visual Studio Code**（推荐，免费下载：https://code.visualstudio.com/）
   - **Notepad++**（简单好用，免费下载：https://notepad-plus-plus.org/）

2. **一个浏览器**：
   - Chrome、Edge、Firefox等任意一个即可

### 可选准备：
- 你的照片（用作头像）
- 你的个人信息

## 💻 如何打开和编辑文件

### 方法1：使用Visual Studio Code（推荐）
1. 下载并安装 Visual Studio Code
2. 右键点击 `personal_homepage.html` 文件
3. 选择"打开方式" → "Visual Studio Code"
4. 文件会在编辑器中打开，可以看到代码

### 方法2：使用记事本
1. 右键点击 `personal_homepage.html` 文件
2. 选择"打开方式" → "记事本"
3. 文件会在记事本中打开

## ❓ 常见问题解答

### Q1: 修改后没有效果怎么办？
**答：** 
1. 确保已经保存文件（Ctrl + S）
2. 在浏览器中按 F5 刷新页面
3. 检查是否修改了正确的位置

### Q2: 修改后没有效果怎么办？
**答：** 
1. 确保已经保存文件（Ctrl + S）
2. 在浏览器中按 F5 刷新页面
3. 检查是否修改了正确的位置

### Q3: 页面显示乱码怎么办？
**答：**
1. 确保文件编码是 UTF-8
2. 在 Visual Studio Code 中，右下角可以看到编码格式
3. 如果不是 UTF-8，点击它选择 "通过编码保存" → "UTF-8"

### Q4: 我不小心删错了代码怎么办？
**答：**
1. 在编辑器中按 `Ctrl + Z` 撤销操作
2. 或者从备份文件中恢复（建议修改前先复制一份备份）

### Q5: 如何在手机上查看效果？
**答：**
1. 将文件传到手机
2. 用手机浏览器打开即可
3. 页面会自动适配手机屏幕

### Q6: 可以改成中文网址吗？
**答：**
可以！将文件名改为 `index.html` 后，可以部署到GitHub Pages等免费托管服务，就有自己的网址了。

## 📱 部署到网上（让别人也能访问）

### 使用 GitHub Pages（免费）

1. **注册 GitHub 账号**
   - 访问 https://github.com
   - 点击 "Sign up" 注册

2. **创建仓库**
   - 登录后点击 "New repository"
   - 仓库名必须是：`你的用户名.github.io`
   - 例如：`zhangsan.github.io`

3. **上传文件**
   - 将 `index.html` 文件拖拽到仓库中上传
   - 提交文件（Commit）

4. **启用 GitHub Pages**
   - 进入仓库的 Settings（设置）
   - 找到 Pages 选项
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "main"
   - 点击 Save

5. **访问你的网站**
   - 等待几分钟
   - 访问 `https://你的用户名.github.io`
   - 例如：`https://zhangsan.github.io`
