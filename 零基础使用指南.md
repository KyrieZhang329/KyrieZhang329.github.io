## 📖 什么是这个文件

这是一个个人主页的网页文件，就像是你的网上名片。它包含：
- 你的照片
- 你的姓名和个人简介
- 你的联系方式
- 你的新闻动态
- 你的工作/学习经历

## 🛠️ 需要准备什么

### 必需品：
1. **一个文本编辑器**（选择其中一个即可）：
   - 记事本（Windows自带，但不推荐）
   - **Visual Studio Code**（推荐，免费下载：https://code.visualstudio.com/）
   - **Notepad++**（简单好用，免费下载：https://notepad-plus-plus.org/）

2. **一个浏览器**：
   - Chrome、Edge、Firefox等任意一个即可

### 可选准备：
- 你的照片（用作头像）
- 你的背景图片（用作顶部背景）

## ✏️ 详细修改步骤

### 📋 步骤1：修改背景图片

**位置：第34-48行**

找到这段代码：
```css
/* 背景图片的网址设置 */
/* 修改说明：将下面url()括号中的网址替换为你想要的图片网址 */
/* 示例1：使用网络图片 url('https://example.com/your-image.jpg') */
/* 示例2：使用本地图片 url('./images/background.jpg') */
background-image: url('https://via.placeholder.com/1920x600');
```

**如何修改：**

#### 方法A：使用网络图片
1. 在网上找到你喜欢的图片
2. 右键点击图片 → "复制图片地址"
3. 将 `url()` 括号中的内容替换为你复制的地址

例如：
```css
background-image: url('https://example.com/my-photo.jpg');
```

#### 方法B：使用本地图片
1. 在 `personal_homepage.html` 文件所在的文件夹创建一个 `images` 文件夹
2. 把你的图片放进 `images` 文件夹，例如命名为 `background.jpg`
3. 修改代码为：
```css
background-image: url('./images/background.jpg');
```

**💡 提示：**
- 背景图片建议使用宽度1920像素以上的图片，这样在大屏幕上也清晰
- 图片格式可以是 .jpg、.png、.jpeg 等

#### 调整背景图片大小

找到第36行：
```css
height: 600px;
```

**如何调整：**
- 数字越大，背景显示越高
- 推荐值：500px - 800px
- 例如想要更高的背景，改成：
```css
height: 700px;
```

**如何修改：**

#### 修改邮箱：

将每个 `https://...` 替换为你的个人主页链接

```html

<li><a href="https://space.bilibili.com/123456" target="_blank">Bilibili</a></li>
```

#### 删除不需要的联系方式：
如果你没有某个社交媒体账号，直接删除整行

例如，删除Twitter：
```html
<!-- 直接删除这一行 -->
```

## ❓ 常见问题解答

### Q1: 修改后没有效果怎么办？
**答：** 
1. 确保已经保存文件（Ctrl + S）
2. 在浏览器中按 F5 刷新页面
3. 检查是否修改了正确的位置

### Q2: 背景图片不显示怎么办？
**答：**
1. 检查图片地址是否正确
2. 如果是本地图片，确保图片和HTML文件在正确的位置
3. 确保图片格式是 .jpg、.png 或 .jpeg

### Q3: 页面显示乱码怎么办？
**答：**
1. 确保文件编码是 UTF-8
2. 在 Visual Studio Code 中，右下角可以看到编码格式
3. 如果不是 UTF-8，点击它选择 "通过编码保存" → "UTF-8"

### Q4: 我不小心删错了代码怎么办？
**答：**
1. 在编辑器中按 `Ctrl + Z` 撤销操作
2. 或者从备份文件中恢复（建议修改前先复制一份备份）

### Q5: 如何在手机上查看效果？
**答：**
1. 将文件传到手机
2. 用手机浏览器打开即可
3. 页面会自动适配手机屏幕

### Q6: 可以改成中文网址吗？
**答：**
可以！将文件名改为 `index.html` 后，可以部署到GitHub Pages等免费托管服务，就有自己的网址了。

## 📱 部署到网上（让别人也能访问）

### 使用 GitHub Pages（免费）

1. **注册 GitHub 账号**
   - 访问 https://github.com
   - 点击 "Sign up" 注册

2. **创建仓库**
   - 登录后点击 "New repository"
   - 仓库名必须是：`你的用户名.github.io`
   - 例如：`zhangsan.github.io`

3. **上传文件**
   - 将 `personal_homepage.html` 重命名为 `index.html`
   - 点击 "uploading an existing file"
   - 拖拽 `index.html` 文件上传

4. **启用 GitHub Pages**
   - 进入仓库的 Settings（设置）
   - 找到 Pages 选项
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "main"
   - 点击 Save

5. **访问你的网站**
   - 等待几分钟
   - 访问 `https://你的用户名.github.io`
   - 例如：`https://zhangsan.github.io`
