西电新生学习导航智能体

XDU-FreshmenAssistant is a locally deployable intelligent Q&A assistant designed for Xidian University freshmen. It focuses on onboarding, course planning, campus resources, competition guidance, and the practical questions that appear in the first semester.

[Code] [Project Proposal PDF] [Demo Video]
Competition Context 2026 XDU Spark Cup (Agent development track)
Target Users Freshmen at Xidian University
Deployment Form Local frontend + local backend + structured knowledge base

Project Positioning

What this project solves

01

Freshmen onboarding

Help new students quickly understand enrollment, courses, learning resources, organizations, and the first-stage campus workflow.

02

Knowledge-guided answering

Use a structured local knowledge base first, and only extend to web information when the question is time-sensitive or local material is insufficient.

03

Local runnable delivery

The project is packaged as a locally runnable front-end and back-end system with one-click startup on Windows.

System Structure

How the project is organized

Frontend Layer

A standalone web interface designed for local use. In the runnable version, it supports text interaction, thinking state, and progressive output.

  • Local static frontend
  • Separated deployment from backend
  • Designed for direct use by freshmen

Backend Layer

A local backend exposing an OpenAI-compatible endpoint, primarily via /v1/chat/completions, with CORS support for front-end/back-end separation.

  • Local health check and service entry
  • Windows-oriented startup scripts
  • Easy integration with agent backends

Knowledge Layer

Standardized knowledge directories separate raw material, curated content, templates, processed data, and indexes for later retrieval iteration.

  • raw source documents
  • curated cleaned knowledge
  • templates and indexes for processing

Run mode

Interaction strategy

The practical chain prioritizes local knowledge. If the problem involves strong timeliness or the local material is not enough, the system can supplement through external information.
